This site is built with Gatsby theme "Clarrise" — a *minimalist blog theme* for writers — suitable for long-form content, travel blog or personal site.

## Installation

Create a new blog from scratch:

```
gatsby new my-travel-blog https://github.com/tacogator/gatsby-themes/gatsby-starter-material-clarisse-blog
```

You can also add the theme to existing Gatsby project:

```
yarn add @tacogator/gatsby-theme-blog-material-clarrise
```
