This site is built with Gatsby theme "Clarrise" — a *minimal blogging theme* for writers — suitable for long-form content, travel blog or personal site.

New to GatsbyJS?  Visit [gatsbyjs.org](https://www.gatsbyjs.org) to learn more.

## How to use this theme

Create a new blog from scratch:

```
gatsby new my-travel-blog https://github.com/tacogator/gatsby-themes/gatsby-starter-material-clarisse-blog
```

You can also add the theme to existing gatsby project:

```
yarn add @tacogator/gatsby-theme-material-clarrise
```
