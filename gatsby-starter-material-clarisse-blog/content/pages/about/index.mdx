**Clarisse** theme fuses together written words and photography.  Suitable for long-form content, travel blog, recipes or general personal site.

![header image](./george-alexandru-novac-unsplash.jpg)
Photo by GEORGE ALEXANDRU NOVAC on [Unsplash](https://unsplash.com/@georgenovac)

## How to use this theme

If starting a new blog from scratch 
```
gatsby create my-travel-blog https://sldldsldl
```

You can also add the theme to existing gatsby project

```
yarn add @tacogator/gatsby-theme-clarrise-starter
```
